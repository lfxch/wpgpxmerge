== Description ==

Addon for WP-GPX-MAPS, allows merging multiple GPX Files for display with WP-GPX-MAPS.
Also includes a Reducer, for shrinking the merged GPX File.

== Installation ==

1. Copy wpgpxmapsmerge to your Plug Folder
2. Activate the Plugin
3. Create your Merged Maps :)
